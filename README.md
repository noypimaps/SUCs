# SUCs
The following repository contains the Philippine's State Univerties and Colleges locations and its 2015-2016 data on student enrollment and government's cost per student.


## Requirements and/or Libraries Used
- [Leaflet](http://leafletjs.com)
- [Leaflet Measure](https://github.com/ljagis/leaflet-measure)
- [Leaflet Marker Cluster](https://github.com/Leaflet/Leaflet.markercluster)
- [Leaflet GeoCSV](https://github.com/joker-x/Leaflet.geoCSV)
- [Jquery](https://jquery.com)
